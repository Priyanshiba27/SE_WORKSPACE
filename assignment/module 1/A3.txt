A) Write the full form and one-line use of: HTTP, HTTPS, FTP, SMTP.

1. HTTP ( Hyper Text Transfer Protocol) – Used for transferring web pages and data between a web browser and a web server.

2. HTTPS (Hyper Text Transfer Protocol Secure) – Used for secure communication between a web browser and a web server using encryption.

3. FTP (File Transfer Protocol) – Used for uploading and downloading files between a computer and a server.

4. SMTP (Simple Mail Transfer Protocol) – Used for sending emails from one mail server to another or from an email client to a mail server.




B) What is the difference between IPv4 and IPv6? Answer in 2 lines.

1. IPv4 uses a 32-bit address format (e.g., 192.168.1.1) and supports about 4.3 billion unique addresses.

2. IPv6 uses a 128-bit address format (e.g., 2001:0db8::1) and supports a vastly larger number of unique addresses.

D) What do these HTTP status codes mean: 200, 301, 404, 500, 403?

1. 200 means OK :- The request was successful, and the server returned the requested page or data.

2. 301 means Moved Permanently :- The requested page has been permanently moved to a new URL.

3. 404 means Not Found :- The requested page or resource does not exist on the server.

4. 500 means Internal Server Error :- Something went wrong on the server while processing the request.

5. 403 means forbidden :- the server understood the request but refuse to allow access. 

 


E) Explain DNS hierarchy (root, TLD, domain, subdomain) with www.google.com as an example.

DNS works like the internet's phonebook. it translate a domain name into an ip address that computers can understand. 

1. Root Domain (.)
The highest level in the DNS hierarchy.
Represented by a dot (.), though it is usually hidden.
Root DNS servers know where to find all Top-Level Domains (TLDs).

Example:
www.google.com. (the final dot represents the root)


2. Top-Level Domain (TLD)
The part immediately below the root.
Common TLDs include .com, .org, .net, .edu, and country codes like .in.

Example:
In www.google.com, the TLD is .com.


3. Domain (Second-Level Domain)
The name registered by an organization or individual under a TLD.

Example:
In www.google.com, google is the domain name.


4. Subdomain
A subdivision of the main domain used to organize services or websites.

Example:
In www.google.com, www is a subdomain.
